Portfolio Website

Here you can see all the work I have done for my own portfolio website, i will show some coding but also the process on how to start and build your own website.

Above you can see almost everything I used to design and style my portfolio website

First I started with a mockup, to visualize what I want my website to look like.
Figma is mostly used for that but you can also use Photoshop,
however figma is more focused on mockup and has the basic feastures which can help a lot.

After making my mockups, I started the basic map structure to order everything. Making sure you dont get lost or lose any files.
This is very important because it can save you a lot of time searching for your codes.

The next step I took was creating all the files I 100% needed, such as: Index.html/php, Style.css and script.js
After that is ofcourse the most obviuous part, coding and styling the website piece by piece.
